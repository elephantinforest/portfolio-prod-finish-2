# 祇園小箱　貴重品管理アプリ
# 概要
* 大事な書類や絶対になくしたくない小物（結婚指輪など）を簡単に管理できるアプリを作成しました。このアプリを使えば、貴重品の場所や情報を一元管理し、紛失や忘れ物の心配を軽減することができます。
* テキストベースの管理ではなく画像ベースで管理するので画面を確認すればすぐに登録した物をみつけれる。
* 登録する際に記入する内容もすくないので簡単に利用できる。
# 作成背景・目的
アプリ開発のきっかけは、ふるさと納税の申込みでした。ワンストップ特例制度の書類を紛失し、申請期限内に発送することができませんでした。その経験から、このような書類の所在管理の難しさ、そして期限切れによる手続きの煩雑さを痛感しました。
膨大な書類を整理整頓し、必要な時に迅速に探し出すことは、現代社会においてもなお大きな課題です。特に、税金控除などの申請には、正確な書類と期限厳守が求められます。私の経験は、多くの人が同様の困難に直面していることを実感させました。
そこで、このような書類の紛失や所在不明を防止するアプリケーション開発を始めました。使いづらいと継続利用しなくなると思うので操作が直感的にに出来るように登録は簡単に出来て、説明をみなくても利用できる簡単な設計を意識して開発しました。

MVCフレームワークの構造を深く理解するために、Laravelなどの既存のフレームワークを使用せず、自らMVCモデルのアプリケーションを一から構築しました。この過程を通じて、MVCアーキテクチャの基本原理や各コンポーネントの役割を詳細に把握し、フレームワークの内部構造とその機能についてより深い知見を得ることができました。
* 今までの所持品管理アプリのテキスト情報で保管場所を保存していているのでいざ探そうとしてもどこに置いたかわからなくなる問題を保管場所の登録方法をかえて解決！
操作は2ステップ。
保管場所を撮影して登録

保管したい物をコメントをつけて登録


既存のアプリでは保管した部屋はわかるけどその部屋のどこに置いたかわからなくなる事が問題だと思いました。そこで所持品の登録方法を変える事でその問題を解決しました。
